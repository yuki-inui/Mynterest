# 機能一覧

## Function1
　予算を設定した上でコンクリートの限界打設数量を求める。   
ex)予算500万円  
	条件)	2万円/m^3 ,最低購入数量0.75m^3,0.25単位以外であれば繰上げで計算する。  
		3.45 ⇨　3.5   
	
## Function2
　残り注文可能数量を出す。   
  ex)  
| 打設数量(m^3) | 残り数量(m^3) | 残り金(万円) |
| -- | -- | -- |
| 0m^3 | 500m^3 | 1000万 |
| 5m^3 | 495m^3 | 990万 |
| : | : | : |
| 495 | 5 | 10万 |
